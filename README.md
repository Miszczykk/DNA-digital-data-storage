# PL

Aby zakodować dowolne dane w DNA, dane są one najpierw konwertowane na system trójkowy. Następnie każda cyfra jest przekształcana w nukleotyd według tabeli kodowania. Wybór kolejnego nukleotydu jest zależny od poprzedniego, co ma na celu zapobieganie powstaniu homopolimerów (ciągów DNA, w których powtarza się ten sam nukleotyd). 

| Poprzedni | 0 | 1 | 2 |
| :--------:|:-:|:-:|:-:|
| A         | C | G | T |
| C         | G | T | A |
| G         | T | A | C |
| T         | A | C | G |

# EN

To encode any data into DNA, the data is first converted into a ternary (base-3) system. Next, each digit is converted into a nucleotide according to a coding table. The selection of the next nucleotide depends on the previous one, which aims to prevent the formation of homopolymers (DNA sequences where the same nucleotide is repeated).

| Previous  | 0 | 1 | 2 |
| :--------:|:-:|:-:|:-:|
| A         | C | G | T |
| C         | G | T | A |
| G         | T | A | C |
| T         | A | C | G |
